# Event Manager

A minimal, basic, yet clean and responsive Event Manager application built with **Next.js**, following the requirements to keep the UI simple and straightforward.

---

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/sahilkhattal09/event-manager.git
   cd event-manager

    Navigate to the app folder:

cd apps/events-app

Install dependencies:

pnpm install

Start the development server:

pnpm dev

Open your browser and go to:

    http://localhost:3000/events
